My Work

Breakout

Breakout is a 1979 arcade game created by Atari. It was the first game to feature a paddle and a ball. The player controls the paddle to bounce the ball and break the bricks. The goal is to break all the bricks without letting the ball fall off the screen.

Space Invaders

Space Invaders is a 1978 arcade game created by Atari. The player controls a space ship and must shoot down a fleet of alien ships. The game ends when all the alien ships are destroyed or when the player ship is destroyed.

Rocke-Paper-Scissors

Rock Paper Scissors is a hand game played between two people. Each player simultaneously forms one of three shapes with an outstretched hand. The possible shapes are rock, paper, and scissors. The game has only three possible outcomes other than a tie.

Memory Game

Memory Game is a game where the player has to remember the pattern of lights that appear on the screen. The player has to repeat the pattern by pressing the buttons in the correct order.
